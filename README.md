# UHI / Business Challenge II

Urban heat island (UHI) pipeline, Sentinel/Landsat notebooks, and related geo features.

## Layout

- `notebooks/` — Jupyter work including `UHI_Unified_Pipeline.ipynb`, Brazil/Chile pipelines, and `business-challenge-2/` (course starter/benchmark notebooks).
- `src/` — Python modules and scripts (`build_uhi_unified_notebook.py`, `main.py`, geo/satellite helpers from Downloads).
- `data/raw/` — Extract text dumps and similar inputs.
- `data/processed/` — Model outputs and submission CSVs.
- `outputs/` — Figures and exports (add as needed).

## Environment

See `requirements.txt`. Use your existing Anaconda base or create a dedicated conda env if you need isolation.
